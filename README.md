# webpack-template
从零开始搭建webpack

使用到的核心组件库：

#### loader

|           名称           |               描述                |
| :----------------------: | :-------------------------------: |
| style-loader、css-loader |              处理css              |
| file-loader、url-loader  |   处理图片(css和js中引入的图片)   |
|   html-withimg-loader    | 处理图片(html文件中img引入的图片) |







#### plugin

|         名称         |                  描述                  |
| :------------------: | :------------------------------------: |
| html-webpack-plugin  |          用于自动生成html文件          |
| clean-webpack-plugin | 每次打包前自动清除打包目录下存在的文件 |
